Simple configuration

You need a Mysql database, as well as an object storage service, which can be obtained from cloud service vendors. If you already have it, configure it now.
mysql
DNS // DNS={user}:{password}@tcp({host}:{port})/haixin?charset=utf8mb4&parseTime=True&loc=Local
tecentCOS
COS // COS=https://%s-{your}.myqcloud.com/
COSSecretID
COSSecretKey

Try to run

$ hixnav
