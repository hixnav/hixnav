# HixNav
 
#### Introduce 
Hixnav supports custom navigation, custom links, cloud disk, cloud image, cloud account management

#### Build

```shell
# windows
$ go build -o hixnav.exe main.go

# linux
$ go build -o hixnav main.go
```

#### Run

```shell
# windows
$ hixnav.exe

# linux
$ hixnav
```

#### Docker

```shell
$ docker pull ghcr.io/hixnav/hixnav:latest
```
